Question-3: What is event delegation in the DOM?

Event Delegation is a technique where a single event listener is added to a parent element to handle events on its child elements using event bubbling. It improves performance, works on dynamic elements, and reduces multiple event listeners. Instead of adding events to each child, the parent listens and detects the target element. This is useful for handling lists, tables, or dynamically added elements

Question-4: What is event bubbling in the DOM?

Event Bubbling is a phase in the DOM event propagation where an event starts from the target element and bubbles up to its parent elements. It allows a parent to handle events triggered by its child elements. Bubbling helps in event delegation, making event handling more efficient.
